## About This Repo

This repository serves as a personal study log and lab journal for the CompTIA Security+ certification. Each lab includes:

- **Objective** — what the lab is meant to teach
- **Tools used** — software, VMs, or platforms involved
- **Steps performed** — a summary of what was done
- **Screenshots / evidence** — stored in each lab's folder
- **Key takeaways** — notes on what I learned or found tricky

> ⚠️ **Disclaimer:** All labs are performed in isolated/sandboxed environments (local VMs, TryHackMe, or Dion's provided lab environment) for educational purposes only. No unauthorized systems are targeted.
